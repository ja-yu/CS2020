```
bandwidth與bps
1.每種通訊頻道可支援在一段時間內能傳輸特定數量的資料
2.描述通訊頻道容量的術語
3.Bandwidth計數單位為bps

REID
1.radio frequency identification
2.經由天線傳輸的無線電訊號
3.收費站的感應器、動物體內的嵌入式晶片

NFC
1.near field communication
2.裝置之間距離小於8英吋，才能進行通訊和完成資料無線傳輸動作
3.應用在信用卡、智慧型手機、車票

蜂巢式網路
1.cellular network
2.行動網路的正式名稱，包括3G、4G、5G
3.3G網路在大多數有提供行動通訊服務的地點，都能提供Internet服務
4.4G、5G網路提供更高速的資料傳輸

群眾外包
1.crowdsourcing
2.運用Internet和群眾的智慧，為了所有人的福祉而共同完成某項計畫或解決某個問題
3.計畫發起人在Internet上公開呼籲(開始召集)，有興趣的人就會參與響應
4.社區計畫、創作計畫、貢獻專長計畫、當地計畫

群眾集資
1.crowdfunding
2.一種特別類型的群眾外包模式
3.眾人在Internet集結，共同替某個計畫、原因或生意點子籌措資金

中繼資料
1.metadata
2.用來描述其他資料的資料
3.相片的中繼資料可能有拍照地點的GPS位置，被張貼的位置，誰張貼的

內容管理系統
1.content management system, CMS
2.適用在建構部落格、大型企業或機構的網站、網路商店
3.許多CMS是開放原始碼，定期提供更新版本、改進功能、外掛程式和主題
4.有利於為數眾多的人員協同作業，不同的人員分別管理網站的一部份網頁
5.能依據人員的工作角色限制存取範圍
6.讓一群使用者共同維護和發行各種內容，常用在網站上

DDoS攻擊
1.distributed denial of service attack
2.利用多台電腦攻擊單一伺服器，傳送給伺服器無法負荷的網路流量，讓它無法提供必要服務
3.通常利用殭屍攻擊

殭屍
1.zombie
2.感染惡意軟體而被攻擊者所控制的裝置
3.一般電腦的擁有者都沒有察覺到自己的系統已經被「殭屍化」

自適應網路設計
1.responsive web design
2.根據顯示螢幕的尺寸自動調整內容版面
3.在小型螢幕使用者不必為了想看到全部版面而左右捲動

搜尋引擎最佳化
1.search engine optimization, SEO
2.讓搜尋引擎更容易找到網站或建立索引
3.可讓網站出現在搜尋結果前面的位置，容易讓使用者點選
```
```
VLAN
1.Virtual Local Area Network
2.網管人員可以藉此透過控制交換器有效分派出入區域網的封包到正確的出入埠
3.降低區域網內大量資料流通時，因無用封包過多導致壅塞的問題，以及提昇區域網的資訊安全保障。
```
