# 重要名詞解釋
```
Von Neumann architecture
1.是一種將程式指令記憶體和資料記憶體合併在一起的電腦設計概念架構。
2.電腦被分成了控制單元、算術邏輯單元(ALU)、記憶體、「輸入設備」（Input）與「輸出設備」（Output）。
3.將儲存裝置與中央處理器(CPU)分開
4.本架構設計出的計算機又稱儲存程式電腦。

虛擬記憶體
1.是作業系統具備的記憶體管理技術
2.將資料暫時儲存在儲存媒體中，直到它能被置換進入RAM為止
3.


POST
1.中文為開機自我檢測
2.是BIOS的一個功能
3.會測試電腦的全部元件是否能正常運作
4.檢測到錯誤時，會在螢幕上顯示錯誤資訊或以警報聲響次數指出檢測到的故障
```

# 簡答題
```
作業系統的功能
1.電腦或裝置的開機和關機
2.管理程序
3.管理記憶體
4.協調工作
5.設定裝置
6.建立Internet連線
7.監控效能
8.提供管理檔案功能
9.更新作業系統軟體
10.監控安全性
11.控制網路存取

Memory hierarchy
1.是在電腦架構下儲存系統階層的排列順序。
2.每一層於下一層相比都擁有較高的速度和較低延遲性，以及較小的容量
3.分成四層: 1.暫存器、2.快取、3.主記憶體、4.磁碟儲存
```

# 負數表達法
```
-39

(1)signed-magniture
step1:取數字的正數部分

-39 ==> 39

step2:將正數部分轉換成7個位元的二進位值

39 ==> 0100111

step3:在最前面加上符號位元，正數為0，負數為1

0100111 ==> 10100111

(2)1's complement
step1:取數字的正數部分

-39 ==> 39

step2:將正數部分轉換成8個位元的二進位值

39 ==> 00100111

step3:將每個位元反相

00100111 ==> 11011000

(3)2's complement
step1:取數字的正數部分

-39 ==> 39

step2:將正數部分轉換成8個位元的二進位值

39 ==> 00100111

step3:將每個位元反相

00100111 ==> 11011000

step4:將反相後二進位值加1

11011000 + 1 ==> 11011001
```
