# img 標籤

### src、width、height 屬性
```
<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>示範圖片</title>
  </head>
  <body>      
    <img src="jp3.jpg" width="30%"><br>	
    <img src="jp3.jpg" width="300" height="200">
    <img src="jp3.jpg" width="150" height="100">	
  </body>
</html>
```
### alt 屬性
```
<!doctype html> 
<html> 
  <head>
    <meta charset="utf-8">
    <title>示範圖片</title>
  </head>
  <body>
    <img src="jp.jpg" width="300" height="200" alt="日本豪斯登堡">
  </body>
</html>
```
# figure、figcaption 標籤
```
<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
	<title>示範標註</title>	
  </head>	
  <body>
    <figure>
      <img src="jp1.jpg">
      <figcaption>日本豪斯登堡</figcaption>
    </figure>
  </body>
</html>
```
