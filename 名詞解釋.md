```
bandwidth與bps
1.每種通訊頻道可支援在一段時間內能傳輸特定數量的資料
2.描述通訊頻道容量的術語
3.Bandwidth基本單位為bps(bits per second)

RFID
1.radio frequency identification
2.經由天線傳輸的無線電訊號
3.通過無線電訊號識別特定目標
4.應用在收費站的感應器、動物體內的嵌入式晶片

NFC
1.near field communication
2.裝置之間距離相距幾公分之內進行通訊和完成資料無線傳輸
3.應用在信用卡、智慧型手機、車票

cellular network
1.行動網路的正式名稱
2.各通訊基地台的訊號覆蓋區域呈六邊形
3.網路類型有3G、4G、5G


crowdsourcing
1.利用Internet和眾人共同完成某個計畫
2.計畫發起人在Internet上召集，有興趣的人參與響應
3.社區計畫、創作計畫、貢獻專長計畫、當地計畫

crowdfunding
1.一種特別類型的群眾外包模式
2.透過群眾的力量來募得資金
3.藉由「購買」或「贊助」的方式以實現計劃、設計或夢想。
4.在一定的時限內，達到事先設定募資的金額目標後即為募資成功，用募到的錢進行計畫

差異:
中文 群眾外包(crowdsourcing ) 群眾募資(crowdfunding )
crowdfunding 是透過群眾的力量來募得資金
crowdsourcing 是想要獲得群眾的想法、知識、技術或是技能


metadata
1.描述資料屬性（property）的資訊
2.協助資料檢索
3.建立的時間與日期、檔案大小等

CMS
1.content management system
2.讓一群使用者共同維護和發行各種內容，常用在網站上
3.能依據人員的工作角色限制存取範圍
4.適用在建構部落格、大型企業或機構的網站、網路商店

DDoS攻擊
1.distributed denial of service attack
2.利用多台電腦攻擊單一伺服器
3.傳送給伺服器無法負荷的偽造包或垃圾封包，讓它無法提供必要服務

botnet
1.感染惡意軟體而被攻擊者所控制的裝置稱為殭屍
2.botnet利用大批殭屍所組成
3.通常蠕蟲病毒也可以被利用組成botnet。

RWD
1.responsive web design
2.根據顯示螢幕的尺寸自動調整內容版面
3.減少使用者進行縮放、平移和捲動等操作

SEO
1.search engine optimization
2.讓搜尋引擎更容易找到網站或建立索引
3.可讓網站出現在搜尋結果前面的位置，容易讓使用者點選

technology addiction
1.中文為科技成癮
2.無法停止使用科技設備
3.利用科技追求快樂
```
```
VLAN
1.Virtual Local Area Network
2.網管人員可以藉此透過控制交換器有效分派出入區域網的封包到正確的出入埠
3.降低區域網內大量資料流通時，因無用封包過多導致壅塞的問題，以及提昇區域網的資訊安全保障。
```
